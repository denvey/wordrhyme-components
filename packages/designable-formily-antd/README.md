# @wordrhyme/designable-formily-antd

Forked from `formily/antd` in [alibaba/designable](https://github.com/alibaba/designable/tree/main/formily/antd) at commit `3e961de947011df6ac5cd092c1084140494f5fc2`.

This package keeps upstream Designable code available under the `@wordrhyme/*` namespace so WordRhyme can adapt the editor without depending on the abandoned upstream package names.

Ant Design/Formily Ant Design dependencies are preserved for migration compatibility. The long-term WordRhyme editor should adapt these surfaces to the shadcn/Formily stack instead of building new product code directly on Ant Design.
